# 一 概述:
## (1)概述:
- kubeadm是个快速创建kubernetes集群的工具, 提供获得一个最小可行和安全集群启动和运行的方式.
- kubeadm应用范围被限制在本地node文件系统和kubernetes API, 被作为高层次工具的组成模块.
- kubeadm也支持其他集群生命周期功能, 例如:bootstrap tokens和集群升级.
